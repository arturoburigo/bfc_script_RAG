# gfipMatricula_buscaGfip
Description: GFIP matrícula
Method: buscaGfip (GET)
Method Description: Listagem de dados de Controle de Modalidades de Encargos INSS e FGTS
Representation Type: GfipFonteDados
Expressions:
- subTipoProcessamento: Subtipo de processamento de cálculo (Type: )
- tipoRecolhimento: Tipo de recolhimento utilizado na geração da GFIP (Type: )
- situacaoEnvioMatricula: Situação do envio da GFIP (Type: )
- gfipMatriculas.matricula.id: Matrícula do empregado (Type: Long)
- tipoProcessamento: Tipo de processamento de cálculo (Type: )
- id: Código da GFIP (Type: Long)
- competencia: Competência (Type: YearMonth)
- gfipMatriculas.id: Id Matrícula GFIP (Type: Long)
- modalidade: Modalidade (Type: )
Types:
## GfipMatriculaDto
- codigoMatricula: Código da matrícula (Type: Long)
- codigoGfip: Código da gfip (Type: Long)
- situacaoEnvioMatricula: Situação do envio da matrícula (Type: )
## MatriculaGfipFonteDados
- id: Matrícula do empregado (Type: Long)
## GfipMatriculaFonteDados
- id: Identificador (Type: Long)
- matricula: Matrícula (Type: MatriculaGfipFonteDados)
## GfipFonteDados
- id: Código da GFIP (Type: Long)
- competencia: Competência (Type: YearMonth)
- tipoRecolhimento: Tipo de recolhimento utilizado na geração da GFIP (Type: )
- situacaoEnvioMatricula: Situação do envio da GFIP (Type: )
- tipoProcessamento: Tipo de processamento de cálculo (Type: )
- subTipoProcessamento: Subtipo de processamento de cálculo (Type: )
- modalidade: Modalidade (Type: )
- gfipMatriculas: Matrícula GFIP (Type: GfipMatriculaFonteDados)
