# cagedMatricula_importa
Description: CAGED matrícula
Method: importa (POST)
Representation Type: CagedMatriculaDto
Types:
## CagedMatriculaDto

- codigoCaged: Código do caged (Type: Long)
- situacaoMatricula: Situação da matrícula (Type: )
- situacaoEnvioMatricula: Situação do envio da matrícula (Type: )
## EstadoMunicipioCaged
- id: Id (Type: Long)
- nome: Nome (Type: String)
- uf: UF (Type: String)
## MunicipioCaged
- id: Id (Type: Long)
- nome: Nome (Type: String)
- estado: Estado (Type: EstadoMunicipioCaged)
## MatriculaMatricula
- id: Código matrícula (Type: Long)
## CagedMatricula

- matricula: Matrícula (Type: MatriculaMatricula)
- situacaoMatricula: Situação da matrícula (Type: )
- situacaoEnvioMatricula: Situação do envio da matrícula (Type: )

- situacaoEnvioMatricula: Situação do CAGED (Type: )
