Method: importa (POST)
Representation Type: TabelaAuxiliarCampoImportacao
Types:
## TabelaAuxiliarAssociacao
- id: Id (Type: Long)
## TabelaAuxiliarCampoImportacao
- id: Id (Type: Long)
- descricao: Descrição do campo (Type: String)
- tipo: Tipo do campo (Type: )
- exibir: Deve exibir o campo? (Type: Boolean)
- tamanho: Tamanho do campo (Type: Integer)
- tabela: Tabela auxiliar (Type: TabelaAuxiliarAssociacao)
- fonteDados:  (Type: Boolean)
## TabelaAuxiliarRegistroSemValores
- id: Id (Type: Long)
- identificador: Identificador do registro (Type: Long)
## TabelaAuxiliarValorSemCampo
- id: Id (Type: Long)
- registro: Registro (Type: TabelaAuxiliarRegistroSemValores)
- valor: Valor do campo (Type: String)
