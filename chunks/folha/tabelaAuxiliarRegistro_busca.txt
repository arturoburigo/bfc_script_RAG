# tabelaAuxiliarRegistro_busca
Description: Tabela auxíliar registro
Method: busca (GET)
Method Description: Listagem de registro das tabelas auxíliares
Representation Type: TabelaAuxiliarRegistro
Expressions:
- id: Id (Type: Long)
- identificador: Identificador do registro (Type: Long)
Types:
## TabelaAuxiliarCampoSemValores
- id: Id (Type: Long)
- identificador: Identificador do campo (Type: String)
- descricao: Descrição do campo (Type: String)
- tipo: Tipo do campo (Type: )
- exibir: Deve exibir o campo? (Type: Boolean)
- tamanho: Tamanho do campo (Type: Integer)
## TabelaAuxiliarValorSemRegistro
- id: Id (Type: Long)
- campo: Informações do campo (Type: TabelaAuxiliarCampoSemValores)
- valor: Valor do campo (Type: String)
## TabelaAuxiliarRegistro
- id: Id (Type: Long)
- identificador: Identificador do registro (Type: Long)
- valores: Valores do registro (Type: TabelaAuxiliarValorSemRegistro)
## TabelaAuxiliarAssociacao
- id: Id (Type: Long)
## TabelaAuxiliarCampoAssociacao
- id: Id (Type: Long)
## TabelaAuxiliarValorAssociacao
- id: Id (Type: Long)

## TabelaAuxiliarRegistroImportacao
- id: Id (Type: Long)
