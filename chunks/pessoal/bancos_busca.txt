# bancos_busca
Description: Bancos
Method: busca (GET)
Method Description: Listagem de bancos
Representation Type: Banco
Expressions:
- numeroBanco: Número (Type: String)
- site: Site (Type: String)
- sigla: Sigla (Type: String)
- associadoFebraban: Associado ao Febraban (Type: Boolean)
- nome: Nome (Type: String)
- cnpj: CNPJ (Type: String)
- id: Id (Type: Long)
Types:
## Banco
- id: Id (Type: Long)
- nome: Nome (Type: String)
- numeroBanco: Número (Type: String)
- sigla: Sigla (Type: String)
- cnpj: CNPJ (Type: String)
- site: Site (Type: String)
- associadoFebraban: Associado ao Febraban (Type: Boolean)
