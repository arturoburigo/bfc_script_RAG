# agenciasBancarias_busca
Description: Agências bancárias
Method: busca (GET)
Method Description: Listagem de agências bancárias
Representation Type: AgenciaBancaria
Expressions:
- bairro.id: Id do bairro (Type: Long)
- numeroEndereco: Número do endereço (Type: String)
- bairro.municipio.id: Id do município (Type: Long)
- banco.id: Id do banco (Type: Long)
- numero: Número (Type: String)
- logradouro.id: Id do logradouro (Type: Long)
- nome: Nome (Type: String)
- id: Id (Type: Long)
- digito: Dígito (Type: String)
- cep: CEP (Type: String)
Types:
## Banco
- id: Id (Type: Long)
- nome: Nome (Type: String)
- numeroBanco: Número (Type: String)
- sigla: Sigla (Type: String)
- cnpj: CNPJ (Type: String)
- site: Site (Type: String)
- associadoFebraban: Associado ao Febraban (Type: Boolean)
## Pais
- id: Id (Type: Long)
- nome: Nome (Type: String)
- sigla2C: Sigla ISO-2 (Type: String)
- sigla3C: Sigla ISO-3 (Type: String)
- sigla3D: Sigla 3D (Type: String)
- codigoBacen: Código no BACEN/IBGE (Type: String)
- codigoEsocial: Código no eSocial (Type: String)
## Estado
- id: Id (Type: Long)
- nome: Nome (Type: String)
- sigla: Sigla (Type: String)
- codigoIbge: Código no IBGE (Type: Integer)
- pais: País (Type: Pais)
## Municipio
- id: Id (Type: Long)
- nome: Nome (Type: String)
- estado: Estado (Type: Estado)
- codigoIbge: Código no IBGE (Type: Long)
- codigoSiafi: Código no SIAFI (Type: Long)
- cep: CEP geral (Type: Long)
## TipoLogradouro
- id: Id (Type: Long)
- descricao: Descrição (Type: String)
- abreviatura: Abreviatura (Type: String)
## Logradouro
- id: Id (Type: Long)
- municipio: Município (Type: Municipio)
- tipoLogradouro: Tipo (Type: TipoLogradouro)

- cep: CEP principal (Type: String)
## Bairro
- id: Id (Type: Long)
