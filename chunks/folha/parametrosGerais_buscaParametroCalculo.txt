# parametrosGerais_buscaParametroCalculo
Description: Parâmetros gerais
Method: buscaParametroCalculo (GET)
Method Description: Exibe a configurações de cálculo do parâmetro geral
Representation Type: ConfiguracaoCalculo
Types:
## TipoEventoMensalFerias
- id: Identificador (Type: Long)
- tipoEvento: Tipo de evento (Type: )
## TipoAfastamentoParametrosGerais
- id: Identificador (Type: Long)
- descricao: Descrição (Type: String)
- classificacao: Classificação (Type: )
## ConfiguracaoCalculo
- id: Identificador (Type: Long)
- pagaUmTercoFeriasIntegralEmFeriasFracionada: Pagamento de ⅓ de férias integral em cálculo de férias fracionadas (Type: Boolean)
- rescicaoAutomaticaVinculoTemporario: Rescisão automática para vínculos temporários (Type: Boolean)
- abonoSemGozo: Abono sem gozo (Type: Boolean)
- descontaFaltasFerias: Descontar faltas sobre férias, inclusive na rescisão (Type: Boolean)
- descontaEmprestimoRescisao: Descontar empréstimos na rescisão (Type: Boolean)
- encerraAfastamentoRescisao: Encerrar afastamentos em aberto na rescisão (Type: Boolean)
- carregaEventosMensaisRescicao: Carregar eventos de processamento mensal na rescisão (Type: Boolean)
- replicaEventosCalculoFeriasParaCalculoMensal: Replicar eventos do cálculo de férias para o cálculo mensal (Type: Boolean)
- carregaEventosMensaisFerias: Carregar eventos variáveis do processamento mensal nas férias (Type: Boolean)
- tiposEventosMensaisFerias: Tipos de eventos de processamento mensal a carregar nas férias (Type: TipoEventoMensalFerias)
- tipoAcaoAlterarFolhaParaPendente: Alterar folha calculadas para pendente ao alterar uma informação que impacta em cálculo de folha? (Type: )
- unidade: Trabalha em (Type: )
- configuracaoDecimoTerceiro: Período aquisitivo de 13º salário (Type: )
- diaPagamentoMensal: Dia para pagamento mensal (Type: Integer)
- tipoAfastamentoFerias: Tipo de afastamento para férias (Type: TipoAfastamentoParametrosGerais)
- isencaoInssPatronalAfastamento: Isenção de INSS Patronal sobre afastamentos (Type: Boolean)
- recolheFgtsAtravesSefip: Recolhe FGTS através da SEFIP (Type: Boolean)
Code Example:
```
fonteParametrosGerais = Dados.folha.v2.parametrosGerais;

dadosParametrosGerais = fonteParametrosGerais.buscaParametroCalculo(campos: "tipoAfastamentoFerias(id, descricao, classificacao), tiposEventosMensaisFerias(id, tipoEvento), id, pagaUmTercoFeriasIntegralEmFeriasFracionada, configuracaoDecimoTerceiro, unidade, tipoAcaoAlterarFolhaParaPendente, replicaEventosCalculoFeriasParaCalculoMensal, carregaEventosMensaisRescicao, isencaoInssPatronalAfastamento, diaPagamentoMensal, abonoSemGozo, rescicaoAutomaticaVinculoTemporario")

percorrer (dadosParametrosGerais) { itemParametrosGerais ->

}
```
