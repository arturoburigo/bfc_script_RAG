# caged_buscaCaged
Description: CAGED matrícula
Method: buscaCaged (GET)
Method Description: Listagem de dados de Controle de Modalidades do CAGED
Representation Type: Caged
Expressions:
- entidade: Entidade (Type: Long)
- cagedMatriculas.matricula.codigoMatricula.id: Código matrícula (Type: Long)
- situacaoEnvioMatricula: Situação do CAGED (Type: )
- cagedMatriculas.situacaoMatricula: Situação da matrícula no CAGED (Type: )
- id: Código do caged (Type: Long)
- dataEnvio: Competência (Type: LocalDate)
Types:
## EstadoMunicipioCaged
- id: Id (Type: Long)
- nome: Nome (Type: String)
- uf: UF (Type: String)
## MunicipioCaged
- id: Id (Type: Long)
- nome: Nome (Type: String)
- estado: Estado (Type: EstadoMunicipioCaged)
## MatriculaMatricula
- id: Código matrícula (Type: Long)
## CagedMatricula
- id: Identificador (Type: Long)
- matricula: Matrícula (Type: MatriculaMatricula)
- situacaoMatricula: Situação da matrícula (Type: )
- situacaoEnvioMatricula: Situação do envio da matrícula (Type: )
## Caged
- id: Código do caged (Type: Long)
- dataEnvio: Competência (Type: LocalDate)
- situacaoEnvioMatricula: Situação do CAGED (Type: )
- entidade: Entidade (Type: Long)
- municipio: Município (Type: MunicipioCaged)
- cagedMatriculas: CAGED matrícula (Type: CagedMatricula)
## CagedMatriculaDto
- codigoMatricula: Código da matrícula (Type: Long)
- codigoCaged: Código do caged (Type: Long)
- situacaoMatricula: Situação da matrícula (Type: )
- situacaoEnvioMatricula: Situação do envio da matrícula (Type: )
Code Example:
```
fonteCaged = Dados.folha.v2.caged;

filtroCaged = "id = 213123 or situacaoEnvioMatricula = ENVIO_VALIDADO"

dadosCaged = fonteCaged.buscaCaged(criterio: filtroCaged,campos: "municipio(estado(id, nome, uf), id, nome), cagedMatriculas(matricula(id), id, situacaoMatricula, situacaoEnvioMatricula), id, dataEnvio, situacaoEnvioMatricula, entidade",ordenacao: "entidade,id asc")

percorrer (dadosCaged) { itemCaged ->

}
```
