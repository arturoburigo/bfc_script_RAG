# mediasVantagens_busca
Description: Médias e vantagens
Method: busca (GET)
Method Description: Listagem de médias e vantagens
Representation Type: MediaVantagemFonteDados
Expressions:
- tipoMediaVantagem:  (Type: )
- configuracaoEvento.codigo:  (Type: Long)
- configuracaoEvento.descricao:  (Type: String)
- configuracaoCalculoMediaVantagem: Calcular a partir de (Type: )
- grupoMediaVantagem: Grupo (Type: )
- exibirEmFolha: Exibir em folha? (Type: )
- id: Id (Type: Long)
Types:
## EventosAssociacaoFonteDados
- id: Id (Type: Long)
- codigo: Código (Type: Long)
- descricao: Descrição (Type: String)
## EventosComposicaoAssociacaoFonteDados
- id: Id (Type: Long)
- configuracaoEvento: Evento de composição (Type: EventosAssociacaoFonteDados)
- periodo: Período (Type: )
- abrangencia: Abrangência (Type: )
- valor: Valor (Type: )
- integralizar: Integralizar? (Type: Boolean)
## MediaVantagemFonteDados
- id: Id (Type: Long)
- configuracaoEvento: Evento (Type: EventosAssociacaoFonteDados)
- tipoMediaVantagem:  (Type: )
- grupoMediaVantagem: Grupo (Type: )
- exibirEmFolha: Exibir em folha? (Type: )
- configuracaoCalculoMediaVantagem: Calcular a partir de (Type: )
- eventosComposicao: Eventos de composição (Type: EventosComposicaoAssociacaoFonteDados)
