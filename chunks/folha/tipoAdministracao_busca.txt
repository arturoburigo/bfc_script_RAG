Description: TipoAdministracao
Method: busca (GET)
Method Description: Listagem de tipos de administração
Representation Type: TipoAdministraçãoDadosRepresentation
Expressions:
- tipo: Tipo de Administração (Type: )
- subteto: Valor do Subteto (Type: BigDecimal)
- naturezaJuridica: Natureza Jurídica (Type: )
- descricao: Descrição (Type: String)

- naturezaJuridica: Natureza Jurídica (Type: )
