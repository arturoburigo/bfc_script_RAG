# logradouros_busca
Description: Logradouros
Method: busca (GET)
Method Description: Listagem de logradouros
Representation Type: Logradouro
Expressions:
- tipoLogradouro.id: Id do tipo de logradouro (Type: Long)
- municipio.id: Id do município (Type: Long)
- id: Id (Type: Long)
- cep: CEP principal (Type: String)
- descricao: Descrição (Type: String)
Types:
## Pais
- id: Id (Type: Long)
- nome: Nome (Type: String)
- sigla2C: Sigla ISO-2 (Type: String)
- sigla3C: Sigla ISO-3 (Type: String)
- sigla3D: Sigla 3D (Type: String)
- codigoBacen: Código no BACEN/IBGE (Type: String)
- codigoEsocial: Código no eSocial (Type: String)
## Estado
- id: Id (Type: Long)
- nome: Nome (Type: String)
- sigla: Sigla (Type: String)
- codigoIbge: Código no IBGE (Type: Integer)
- pais: País (Type: Pais)
## Municipio
- id: Id (Type: Long)
- nome: Nome (Type: String)
- estado: Estado (Type: Estado)
- codigoIbge: Código no IBGE (Type: Long)
- codigoSiafi: Código no SIAFI (Type: Long)
- cep: CEP geral (Type: Long)
## Bairro
- id: Id (Type: Long)
- nome: Nome (Type: String)
- municipio: Município (Type: Municipio)
- zonaRural: Zona rural (Type: Boolean)
## TipoLogradouro
- id: Id (Type: Long)
- descricao: Descrição (Type: String)
- abreviatura: Abreviatura (Type: String)
## Logradouro
- id: Id (Type: Long)
- municipio: Município (Type: Municipio)
- tipoLogradouro: Tipo (Type: TipoLogradouro)
- descricao: Descrição (Type: String)
- cep: CEP principal (Type: String)

fonteLogradouros = Dados.pessoal.v2.logradouros;
