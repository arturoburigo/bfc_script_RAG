# municipios_busca
Description: Municípios
Method: busca (GET)
Method Description: Listagem de municípios
Representation Type: Municipio
Expressions:
- municipio.estado.siglaEstado: Sigla do estado (Type: String)
- estado.pais.id: Id do pais (Type: Long)
- estado.id: Id do estado (Type: Long)
- codigoIbge: Código no IBGE (Type: Long)
- nome: Nome (Type: String)
- codigoSiafi: Código no SIAFI (Type: Long)
- id: Id (Type: Long)
- cep: CEP geral (Type: Long)
Types:
## Pais
- id: Id (Type: Long)
- nome: Nome (Type: String)
- sigla2C: Sigla ISO-2 (Type: String)
- sigla3C: Sigla ISO-3 (Type: String)
- sigla3D: Sigla 3D (Type: String)
- codigoBacen: Código no BACEN/IBGE (Type: String)
- codigoEsocial: Código no eSocial (Type: String)
## Estado
- id: Id (Type: Long)
- nome: Nome (Type: String)
- sigla: Sigla (Type: String)
- codigoIbge: Código no IBGE (Type: Integer)
- pais: País (Type: Pais)
## Municipio
- id: Id (Type: Long)
- nome: Nome (Type: String)
- estado: Estado (Type: Estado)
- codigoIbge: Código no IBGE (Type: Long)
- codigoSiafi: Código no SIAFI (Type: Long)
- cep: CEP geral (Type: Long)
Code Example:
```
fonteMunicipios = Dados.pessoal.v2.municipios;

filtroMunicipios = "estado.id = null"

dadosMunicipios = fonteMunicipios.busca(criterio: filtroMunicipios,campos: "id, nome, codigoIbge, cep, codigoSiafi, estado(nome, id, sigla)",ordenacao: "nome asc")

percorrer (dadosMunicipios) { itemMunicipios ->

}
```
